只有

Simplified: 只有

Traditional: 只有

Pinyin: zhi3 you3

only have ...

(used in combination with 才[cai2]) it is only if one ... (that one can ...) (Example: 只有通過治療才能痊愈|只有通过治疗才能痊愈[zhi3 you3 tong1 guo4 zhi4 liao2 cai2 neng2 quan2 yu4] "the only way to cure it is with therapy")

(used to indicate that one has no alternative) one can only (do a certain thing) (Example: 只有屈服[zhi3 you3 qu1 fu2] "the only thing you can do is give in")
